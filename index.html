2. Music Visualiser

A Java Application for Music Visualization Through Amplitude, Frequency, Lerping and 2-D Java Graphics focus Through OOP Approach.

Music Visualiser

Category - JAVA APP

Project Brief:

The program that I have created is a music visualiser that draw shapes/animation on the screen by reacting to the sound input which is an audio. I have use the Minim libary for the audio to be able to play the sound both Mono and stereo. I have used the processing libraries for the draw method to draw shape with a more OOP approach. A lerped function is included to draw the shapes smoothly while reacting to the music to control the size of the shape. The visual.java have abstract classes that are being called in the visualiser.java for the audio. In the visualiser.java, inheritance is used to create a new class from an existing class. For example the draw() method.

  • Tools - Visual Studio Code
  • Languages - JAVA
